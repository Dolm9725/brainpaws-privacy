2. Allgemeines zur Datenverarbeitung

Die App BrainPaws verarbeitet personenbezogene Daten nur insoweit, wie dies für die Bereitstellung der Funktionen der App erforderlich ist. Nachfolgend wird erläutert, welche Daten zu welchem Zweck verarbeitet werden.

3. Lokal gespeicherte Daten

BrainPaws speichert folgende Daten lokal auf dem Gerät des Nutzers:

  • Hundeprofile (Name, optional Profilbild)
  • Trainingsfortschritte, Bewertungen und Einträge
  • Punkte, Freischaltungen und Streaks
  • Individuelle Einstellungen (Sprache, Benachrichtigungen)
  • Optional ausgewählte Bilder für Hundeprofile

Diese Daten werden ausschließlich lokal auf dem Gerät gespeichert und nicht automatisch an externe Server übertragen.

4. Firebase-Dienste (Google)

BrainPaws verwendet Dienste von Google Firebase. Dabei können Daten an Server von Google LLC, 1600 Amphitheatre Parkway, Mountain View, CA 94043, USA übertragen werden.

4.1 Firebase Analytics

BrainPaws verwendet Firebase Analytics zur anonymisierten Analyse der App-Nutzung. Dabei werden unter anderem folgende Daten erfasst:

  • Anonymisierte Nutzungsereignisse (z. B. aufgerufene Screens)
  • Geräteinformationen (Modell, Betriebssystemversion)
  • Sitzungsdauer und -häufigkeit
  • Eine anonymisierte Instanz-ID

Es werden keine Namen, E-Mail-Adressen oder andere direkt personenidentifizierbare Daten an Firebase Analytics übermittelt. Die Daten dienen ausschließlich der Verbesserung der App-Qualität und -Stabilität.

Weitere Informationen: firebase.google.com/support/privacy

4.2 Firebase Crashlytics

BrainPaws verwendet Firebase Crashlytics zur automatischen Erkennung und Analyse von App-Abstürzen. Im Fehlerfall werden folgende Daten übermittelt:

  • Stack Traces und Fehlermeldungen zum Zeitpunkt des Absturzes
  • Geräteinformationen (Modell, Betriebssystem, App-Version)
  • Eine anonymisierte Instanz-ID
  • Uhrzeit des Absturzes

Diese Daten werden ausschließlich zur Fehlerbehebung genutzt und enthalten keine Trainingsdaten, Hundeprofile oder sonstige nutzerspezifische Inhalte.

Weitere Informationen: firebase.google.com/products/crashlytics

Die Nutzung von Firebase-Diensten erfolgt auf Grundlage des berechtigten Interesses gemäß Art. 6 Abs. 1 lit. f DSGVO an der Bereitstellung einer stabilen und nutzerfreundlichen App. Google LLC ist unter dem EU-US Data Privacy Framework zertifiziert, womit ein angemessenes Datenschutzniveau gewährleistet wird.

5. In-App-Käufe

BrainPaws bietet optionale kostenpflichtige Abonnements (BrainPaws Journey) an. Die Abwicklung von Zahlungen erfolgt ausschließlich über den Google Play Store. BrainPaws selbst verarbeitet keine Zahlungsdaten. Es gelten die Datenschutzbestimmungen von Google.

6. Zugriff auf Fotos / Medien

Wenn Nutzer ein Bild für ein Hundeprofil auswählen, greift BrainPaws nur nach ausdrücklicher Auswahl durch den Nutzer auf die entsprechenden Mediendaten des Geräts zu. Das Bild wird ausschließlich lokal in der App gespeichert.

7. Push-Benachrichtigungen

BrainPaws kann optionale lokale Push-Benachrichtigungen senden (z. B. Streak-Erinnerungen). Diese werden lokal auf dem Gerät geplant und erfordern keine Übertragung von Daten an externe Server. Die Benachrichtigungen können jederzeit in den Geräteeinstellungen deaktiviert werden.

8. Keine Weitergabe an Werbenetzwerke

BrainPaws gibt keine personenbezogenen Daten an Werbenetzwerke, Social-Media-Plattformen oder sonstige kommerzielle Dritte weiter. Die App enthält keine Werbung.
