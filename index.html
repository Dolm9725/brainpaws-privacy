2. Allgemeines zur Datenverarbeitung

Die App BrainPaws verarbeitet personenbezogene Daten nur insoweit, wie dies für die Bereitstellung der Funktionen der App erforderlich ist.

3. Lokal gespeicherte Daten

BrainPaws speichert Daten lokal auf dem Gerät des Nutzers, insbesondere:

  • Hundeprofile
  • Trainingsfortschritte
  • Punkte, Freischaltungen und Streaks
  • individuelle Einstellungen
  • optional ausgewählte Bilder für Hundeprofile

Diese Daten werden nicht automatisch an externe Server übertragen.

4. Zugriff auf Fotos / Medien

Wenn Nutzer ein Bild für ein Hundeprofil auswählen, greift BrainPaws nur nach ausdrücklicher Auswahl durch den Nutzer auf die entsprechenden Mediendaten des Geräts zu.

5. Keine automatische Weitergabe an Dritte

Nach aktuellem Stand werden durch BrainPaws keine personenbezogenen Daten automatisiert an Dritte, externe Analyseanbieter oder Werbenetzwerke weitergegeben.
